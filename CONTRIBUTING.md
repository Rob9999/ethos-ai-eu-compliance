# Contributing to EthosAI® EU Compliance

Beiträge willkommen! Bitte beachten Sie:

## Regeln

1. **CLA** — Alle Beiträge erfordern die Unterzeichnung unseres
   [CLA](.github/CLA.md).
2. **CC-BY 4.0** — Alle Inhalte werden unter CC-BY 4.0 veröffentlicht.
3. **Keine internen Details** — Konzeptebene: was, nicht wie.
4. **SPDX-Header** — Jede Datei beginnt mit:
   ```
   <!-- SPDX-License-Identifier: CC-BY-4.0 -->
   <!-- Copyright (c) 2024-2026 Robert Alexander Massinger -->
   ```
5. **Markenrecht** — „EthosAI®" mit Trademark bei erster Erwähnung.

## Pull Requests

1. Fork → Feature-Branch
2. Dokument schreiben/überarbeiten
3. Sicherstellen: keine internen Details, korrekter SPDX-Header
4. PR mit Beschreibung erstellen

Fragen? GitHub Issues nutzen.
